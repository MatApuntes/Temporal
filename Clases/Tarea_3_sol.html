Listas¶

De manera intuitiva, podemos considerar el concepto de lista como una enumeracin. Por ejemplo, cuando vamos al sper podemos crear una lista de las cosas que queremos comprar, digamos

  • Refrescos
  • Botanas
  • Leche
  • etctera

De tal manera, la lista anterior se puede nombrar como lista del sper y sus elementos son productos, en este caso, alimenticios. Decimos de lo anterior que el elemento Refrescos de la lista ocupa la primer posicin, que dentro de Python se le asigna el ndice o lugar 0; las Botanas ocupan el segundo lugar de nuestra lista lista del sper, que Python le asigna el ndice 0, y as sucesivamente. De manera totalmente anloga podemos definir una lista de nmeros, digamos

lista de nmeros: 1, 2, 3 ,4

donde, de nuevo, el primer elemento de la lista, el cual es el 1, Python le asigna la posicin o ndice 0 referente a dicha lista; tambin, podemos decir que el nmero 4 ocupa la posicin nmero 3 de la lista.

Ahora bien, si queremos declarar una lista en Python ocuparemos corchetes, donde dentro de estos corchetes irn los elementos de la lista en cuestin separados por comas. Las listas en Python admiten datos de diferentes tipos, esto es, podemos definir una lista cuyos elementos sean de la clase int, u otros de la clase float, asimismo, podemos declarar que algunos de sus elementos sean cadenas de texto. Diagramas de dispersión¶

Los diagramas de dispersin nos sirven para buscar o visualizar posibles relaciones entre dos variables. Por ejemplo, podemos considerar ahora una lista de nmeros que corresponda a la edad de ciertos nios, la cual llamaremos edad_nums; y consideraremos otra lista que corresponda a sus estaturas, la cual llamaremos altura_nums:

In [15]:
# puntos correspondientes las edades
edad_nums = [4, 4, 6, 8, 8, 5, 2, 7, 9]

# puntos correspondientes a las alturas
# correspondientes a cm
altura_nums = [100, 99.8, 112.9, 123.05, 124.4, 105.5, 85.5, 119.3, 127.7]

Es claro que dichos valores s se relacionan pues la altura depende de la edad, al menos en los primeros 10 aos de vida. Luego, podemos graficar dichos puntos, es decir los puntos de coordenadas $(edad, altura)$ en un diagrama de dispersin (scatter plot o tambin llamado diagrama de puntos) utilizando la funcin plt.scatter() como sigue
