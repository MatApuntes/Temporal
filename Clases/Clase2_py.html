Introducción¶

En esta clase veremos el concepto de condicional en Python, el cual nos permitir realizar una accon si cierta condicin es verdadera, en otras palabras si cierta variable boolena posee el valor de True; y se relizar tambin una accin en dado caso de que la variable boleana tenga el valor de False.

Asimismo, veremos tambin el concepto de bucles, en el cual cierta accin o acciones sern ejecutadas repetidamente cierto nmero de veces o hasta que cierta condicin se cumpla.

Retomando la idea del inicio de la clase, bsicamente un condiconal representa ciertas instrucciones a ejecutar si una condicin se cumple o es verdadera, asimismo, ciertas isntrucciones sern ejecutadas cuando la condicin sea falsa. Por ejemplo

Elif¶

Cuando es necesario que ocupemos varias instrucciones if, o en otras palabras cuando haya muchos casos que considerar para un if (como en el ejemplo anterior), una alternativa ms amigable es utilizar la clusula elif, la cual se puede interpretar como: si no se cumple la condicin anterior, y si...

Por ejemplo, del cdigo anterior tendramos bsicamente

  • si el monto es mayor a 5000 da el descuento del 20%
  • si no, y si el monto es mayor a 2000 pero menor a 5000 da el descuento del 15%
  • Luego, en este caso si conviene ya agregar un else: si no, entonces da el descuento del 10%

En cdigo tenemos

    monto_final = monto - monto * 0.2

# monto mayor o igual a 2000 y menor a 5000

La ventaja es que considerar if y elif trae consigo la agrupacin de todo el cdigo involucrado, y no deja aislados los if como vimos al principio. De esta manera tenemos mayor exactitud en cuanto a las condiciones o casos que estamos manejando.

Bucle for ¶

Un bucle es un concepto, dentro de la programacin, en el cual se estarn repitiendo algunas instrucciones cierto nmero de veces, o mientras una condicin sea verdadera. En particular, el bucle for corresponde a la repeticin de instrucciones un determinado nmero de veces. Por ejemplo, podemos programar lo siguiente:

  • imprime "Hola" cinco veces

o equivalantemente:

  • ejecuta print("Hola") cinco veces

donde en esta ltima equivalencia vemos que una instruccin se est repitiendo (entrando en bucle) un nmero especifico de veces.

donde el bucle for terminar una vez que la instruccin haya sido ejecutada 5 veces. Lo anterior lo implementamos en cdigo como:

Bucle while ¶

En el caso del bucle while, ste es un bucle que se ejecutar siempre y cuando una cierta condicin contine siendo verdadera

y dicho bucle parar slo cuando la condicin sea falsa. La estructura bsica del bucle while es:

while <<condicion verdadera>>:
    instruccion a realizar mientras
    la condicion sea verdadera

Por ejemplo, podemos ejecutar print("Hola") mientras que una variable x sea mayor o igual a 0

# declaramos la variable x inicializandola en 4
x = 4
# declaramos el bucle while
while x >= 0:
    print("Hola")
    # Una vez que se ha impreso la cadena "hola"
    # el bucle while vuelve a evaluar la veracidad
    # de la condicion

El bucle anterior es un bucle infinito pues, como declaramos que x tiene el valor de 4 y en ningn momento hemos cambiado este valor de x, entonces x continuar valiendo 4, de tal manera, como 4 siempre es mayor o igual a cero, el bucle anterior ser un bucle infinito, pues la condicin del while siempre ser verdadera.

Por ello, es comn escribir dentro de nuestro bucle while alguna instruccin para que, en cierto momento, la condicin dada sea falsa. Por ejemplo, en el cdigo anterio podemos restar una unidad al valor de la x cada que se ejecute una iteracin del bucle:

# declaramos la variable x inicializandola en 4

# declaramos el bucle while

    print("Hola")
    # Restamos una unidad al valor actual de la x
    x -= 1

  • Primera validacin de la condicin: se vlida si x >= 0, como x tiene el valor de 4, entonces la condicin del while es verdadera, as, se ejecuta entonces la instruccin print("Hola"). Despus, se ejecuta la instruccin x -= 1 con lo cual cambiamos el valor de la x de 4 a 3.
  • Segunda validacin de la condicin: se vlida si x >= 0, como x tiene el valor de 3, entonces la condicin del while es verdadera, as, se ejecuta entonces la instruccin print("Hola"). Despus, se ejecuta la instruccin x -= 1 con lo cual cambiamos el valor de la x de 3 a 2.
  • Tercera validacin de la condicin: se vlida si x >= 0, como x tiene el valor de 2, entonces la condicin del while es verdadera, as, se ejecuta entonces la instruccin print("Hola"). Despus, se ejecuta la instruccin x -= 1 con lo cual cambiamos el valor de la x de 2 a 1.
  • Cuarta validacin de la condicin: se vlida si x >= 0, como x tiene el valor de 1, entonces la condicin del while es verdadera, as, se ejecuta entonces la instruccin print("Hola"). Despus, se ejecuta la instruccin x -= 1 con lo cual cambiamos el valor de la x de 1 a 0.
  • Quinta validacin de la condicin: se vlida si x >= 0, como x tiene el valor de 0, entonces la condicin del while es verdadera, as, se ejecuta entonces la instruccin print("Hola"). Despus, se ejecuta la instruccin x -= 1 con lo cual cambiamos el valor de la x de 0 a -1.
  • Sexta validacin de la condicin: se vlida si x >= 0, como x tiene el valor de -1, entonces la condicin del while es falsa, de tal manera se interrumpe el bucle y ya no se ejecutan las instrucciones.

Por lo tanto, vemos que se imprime cinco veces el mensaje "hola", pues el bucle while tuvo 5 iteraciones hasta que la condicin dada fue falsa.
