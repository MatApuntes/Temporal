Listas ¶

De manera intuitiva, podemos considerar el concepto de lista como una enumeracin. Por ejemplo, cuando vamos al sper podemos crear una lista de las cosas que queremos comprar, digamos

  • Refrescos
  • Botanas
  • Leche
  • etctera

De tal manera, la lista anterior se puede nombrar como lista del sper y sus elementos son productos, en este caso, alimenticios. Decimos, de lo anterior, que el elemento Refrescos de la lista ocupa la primer posicin, que dentro de Python se le asigna el ndice o lugar 0; las Botanas ocupan el segundo lugar de nuestra lista lista del sper, que Python le asigna el ndice 0, y as sucesivamente. De manera totalmente anloga podemos definir una lista de nmeros, digamos

lista de nmeros: 1, 2, 3 ,4

Ahora bien, si queremos declarar una lista en Python ocuparemos corchetes, donde dentro de estos corchetes irn los elementos de la lista en cuestin separados por comas. Función len()¶

Por otro lado, podemos conocer cuntos caracteres tiene un cadena de texto. Por ejemplo, si consideramos la cadena "Hola", tenemos que, en este caso, dicha cadena tiene en total 4 letras, por lo cual diremos que la longitud de "Hola" es de 4; tambin, podemos ver que la longitud de la cadena "Hola Mundo" es de 10, pues en este caso el espacio en blanco cuenta como un caracter.

En Python podemos utilizar la funcin len() la cual, en particular, nos dir la longitud de una cadena de texto. Veamos:

In [15]:
msj = "Hola"
msj2 = "Hola Mundo"
print(f"Longitud de la cadena msj: {len(msj)}\nLongitud de la \
cadena msj2: {len(msj2)}")

Longitud de la cadena msj: 4
Longitud de la cadena msj2: 10

Asimismos, podemos ocupar esta funcin para conocer la longitud de listas:

Método Split()¶

Mediante este mtodo podemos dividir cadenas de texto palabra por palabra, donde en automtico se crear una lista cuyos elementos son, justamente, las palabras obtenidas de la cadena de texto resultado de la fragmentacin. Por ejemplo, de la cadena "Hola Mundo", por medio del mtodo split() conseguiremos la lista ["Hola" , "Mundo"]. El mtodo split() requiere que especifiquemos el separador de palabras para la fragmentacin a realizar. Por ejemplo, el separador para el caso de "Hola Mundo" ser el espacio en blanco " ", el cual permite poder considerar de manera separada las cadena "Hola" y "Mundo". As, escribiremos entonces, intuitivamente, split(" ").

Otro ejemplo, consideremos la cadena "Hola,mundo,qu hace,:D". Podemos separar cada una de las palabras de dicha cadena tomando como referencia las comas, esto es, separaremos cada palabra de la cadena utilizando como separador ",". Con lo cual obtendremos la lista de palabras ['Hola', 'mundo', 'qu hace', ':D']

Veamos los ejemplos anteriores en accin
